Seletores CSS

Os seletores CSS são utilizados para direcionar elementos específicos em uma página da web e aplicar estilos a eles.

Tipo de Seletor Utilizado? Comentário
Seletor de Elemento done Seleciona elementos com base em seu tipo (tag), como `

` ou `

`. É amplamente utilizado em todas as páginas HTML.
Seletor de Classe done Seleciona elementos com base em sua classe atribuída. É comumente usado para aplicar estilos a elementos específicos com a mesma classe.

Combinadores de Seletores

Combinador Utilizado? Comentário
Descendente (espaço) done Seleciona um elemento que é descendente de outro elemento especificado. É comumente usado para aplicar estilos a elementos filhos de um elemento pai específico.
Seletor de Filho Direto (>) done Seleciona um elemento que é um filho direto de outro elemento especificado. É útil quando você deseja estilizar apenas os filhos diretos de um elemento pai.
