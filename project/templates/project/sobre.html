Selenium: Automação

O script automatiza a interação com uma página web usando o Selenium e o navegador Chrome. Em resumo:

  • Abre o navegador e acessa uma URL específica.
  • Clica em botões para navegar pela página.
  • Preenche campos de login com usuário e senha.
  • Submete o formulário de login.
  • Navega por diferentes partes da página.
  • Preenche campos de criação de artigo.
  • Submete o formulário de criação de artigo.
  • Fecha o navegador.

Estes passos automatizam a interação com a página sem a necessidade de intervenção manual.
